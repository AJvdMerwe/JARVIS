# Install: pip install vllm     (CUDA 12+, PyTorch required)
# Managed: bash start.sh vllm   (starts server + app)
# When LLM_BACKEND=vllm there is NO automatic fallback to Ollama.

# vllm>=0.4.2

#
# Installation:
#   pip install sglang[all]
#
# SGLang is OPTIONAL on CPU/Mac setups. Ollama is the default.
# On GPU servers, SGLang provides significantly higher throughput.
# Uncomment the line below on GPU servers:
# sglang[all]>=0.3.0
#
# Note: When LLM_BACKEND=sglang there is NO automatic fallback to Ollama.
#       If SGLang is not running, the application will fail with a clear error.
